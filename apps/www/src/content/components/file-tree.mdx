## Examples

### Flat

A single-level list — top-level files only, no nested directories.

<Example code={`<FileTree>
- README.md
- package.json
- astro.config.mjs
- tsconfig.json
</FileTree>`} contentClass="w-full">
  <div class="w-full">
    <FileTree>
      <ul>
        <li>README.md</li>
        <li>package.json</li>
        <li>astro.config.mjs</li>
        <li>tsconfig.json</li>
      </ul>
    </FileTree>
  </div>
</Example>

### Nested

Trailing `/` marks a directory; nest a list inside the row to add children.

<Example code={`<FileTree>
- src/
  - components/
    - Header.astro
    - Footer.astro
  - pages/
    - index.astro
</FileTree>`} contentClass="w-full">
  <div class="w-full">
    <FileTree>
      <ul>
        <li>
          src/
          <ul>
            <li>
              components/
              <ul>
                <li>Header.astro</li>
                <li>Footer.astro</li>
              </ul>
            </li>
            <li>
              pages/
              <ul>
                <li>index.astro</li>
              </ul>
            </li>
          </ul>
        </li>
      </ul>
    </FileTree>
  </div>
</Example>

### Highlight and placeholder

Wrap an entry in `**...**` to highlight it. Use `...` (on its own row) for an elided placeholder.

<Example code={`<FileTree>
- src/
  - components/
    - **Header.astro**
    - Footer.astro
    - ...
  - pages/
    - ...
</FileTree>`} contentClass="w-full">
  <div class="w-full">
    <FileTree>
      <ul>
        <li>
          src/
          <ul>
            <li>
              components/
              <ul>
                <li><strong>Header.astro</strong></li>
                <li>Footer.astro</li>
                <li>...</li>
              </ul>
            </li>
            <li>
              pages/
              <ul>
                <li>...</li>
              </ul>
            </li>
          </ul>
        </li>
      </ul>
    </FileTree>
  </div>
</Example>
