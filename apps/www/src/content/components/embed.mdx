---
title: Embed
tagline: "Iframe wrapper with ratio lock and sensible defaults for video, maps, and embeds."
props:
  - name: src
    type: string
    required: true
    description: URL to embed inside the iframe.
  - name: title
    type: string
    required: true
    description: Accessible name for the iframe.
  - name: ratio
    type: string
    defaultValue: '"16/9"'
    description: 'CSS aspect ratio (e.g. "16/9", "4/3", "1/1").'
  - name: allow
    type: string
    description: Permissions policy passed to the iframe.
  - name: allowfullscreen
    type: boolean
    description: Allow the embedded content to enter fullscreen.
---

### Custom ratio

Pass any CSS aspect-ratio string — useful for square previews or ultrawide content.

  src="https://example.com/preview"

    <Embed src="about:blank" title="Square preview" ratio="1/1" />
