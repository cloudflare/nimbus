---
title: LinkCard
tagline: "Link-styled card with an optional leading icon, title, and description. Title and icon shift to primary on hover."
props:
  - name: title
    type: string
    required: true
    description: Heading rendered as the card label.
  - name: description
    type: string
    description: Optional subtitle below the title.
  - name: href
    type: string
    required: true
    description: Target URL — the entire card becomes the link.
  - name: icon
    type: string
    description: Optional Iconify icon name (e.g. `ph:rocket-launch`) rendered before the title.
---
