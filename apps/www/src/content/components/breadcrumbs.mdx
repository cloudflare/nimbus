---
title: Breadcrumbs
tagline: "Hierarchical trail to the current page — collapses long paths with an ellipsis."
props:
  - name: items
    type: "Array<{ label: string; href: string }>"
    required: true
    description: Trail items in document order. The last entry is rendered as the current page.
  - name: maxVisible
    type: number
    defaultValue: "4"
    description: Cap on visible items before middle entries collapse into an ellipsis.
---
